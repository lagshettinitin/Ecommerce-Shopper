# Shopper - Fashion E-Commerce Platform

## Overview
Shopper is a full-stack fashion e-commerce platform that allows users to browse products, add items to the cart, place orders, and manage their accounts.

## Features
- User Authentication
- Product Search and Filtering
- Shopping Cart
- Secure Checkout
- Order Management
- Responsive Design

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Database
- MongoDB

## Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/shopper.git
```

### Install Dependencies

Frontend:

```bash
cd frontend
npm install
npm run dev
```
## Environment Variables

Create a `.env` file and add:

```env
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
